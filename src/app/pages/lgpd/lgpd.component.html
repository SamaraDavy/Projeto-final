Política de Privacidade e LGPD

Saiba como o DevPath utiliza e protege seus dados.

1. Sobre esta política

O DevPath respeita a privacidade dos usuários e segue os princípios da Lei Geral de Proteção de Dados Pessoais, conhecida como LGPD.

2. Dados coletados

Durante o cadastro, o DevPath poderá armazenar informações como:

  • Nome do usuário;
  • Nome de acesso;
  • Senha cadastrada;
  • Data de criação da conta;
  • Anotações e conteúdos registrados na plataforma.

3. Utilização dos dados

Os dados são utilizados apenas para permitir o acesso à plataforma, personalizar a experiência do usuário e salvar suas informações de estudo.

4. Armazenamento

Por se tratar de um projeto acadêmico sem servidor ou banco de dados, as informações são armazenadas localmente no navegador do usuário, utilizando o LocalStorage.

5. Compartilhamento de dados

O DevPath não compartilha, comercializa ou envia os dados cadastrados para terceiros.

6. Direitos do usuário

O usuário pode acessar, alterar ou excluir suas informações armazenadas no sistema a qualquer momento.

7. Consentimento

Ao marcar a opção de aceite durante o cadastro, o usuário declara que leu e concorda com esta Política de Privacidade.
