# 🚀 DevPath

## Sobre o Projeto

O DevPath é uma aplicação web desenvolvida em Angular com o objetivo de auxiliar estudantes e profissionais em transição para a área de tecnologia a organizarem seus estudos, registrarem anotações e acompanharem sua evolução de aprendizado.

O sistema permite o gerenciamento de conhecimentos, criação de anotações e personalização do perfil do usuário, proporcionando uma experiência simples e intuitiva para o acompanhamento da jornada de estudos.

---

## Funcionalidades

* Cadastro de usuários
* Login e logout
* Controle de acesso com autenticação
* Dashboard com resumo das informações
* Cadastro e gerenciamento de conhecimentos
* Criação e exclusão de anotações
* Perfil do usuário personalizável
* Armazenamento local utilizando LocalStorage
* Adequação à LGPD por meio do aceite no cadastro

## Como Executar o Projeto

1. Clone o repositório:

```bash
git clone https://github.com/SamaraDavy/Projeto-final.git
```

2. Acesse a pasta do projeto:

```bash
cd Projeto-final
```

3. Instale as dependências:

```bash
npm install
```

4. Execute a aplicação:

```bash
ng serve
```

5. Abra o navegador:

```text
http://localhost:4200
```
